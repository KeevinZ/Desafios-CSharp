## 📖 Sobre

Este repositório reúne os desafios desenvolvidos durante o curso **C# para Iniciantes**, oferecido pelo **Programa FDevs**.

O objetivo desses exercícios é praticar os principais conceitos da linguagem C#, como:

- Estruturas de repetição
- Entrada e saída de dados
- Operadores matemáticos
- Estruturas condicionais
- Lógica de programação

---

## 📂 Desafios

### 🔢 Desafio 1 - Soma de 1 a 10

Desenvolver um programa que utilize uma estrutura de repetição `for` para calcular a soma dos números de **1 até 10**, exibindo o resultado no console.

**Conceitos praticados**

- `for`
- Variáveis
- Operadores aritméticos
- Acumuladores

---

### ⚖️ Desafio 2 - Calculadora de IMC

Criar um programa que solicite:

- Peso (kg)
- Altura (m)

Em seguida, calcular o **Índice de Massa Corporal (IMC)** utilizando a fórmula:

```text
IMC = peso / (altura × altura)
```

Após o cálculo, o programa informa a classificação do IMC.

**Conceitos praticados**

- Entrada de dados
- Conversão de tipos
- Operações matemáticas
- Estruturas condicionais (`if/else`)

---

### 🧮 Desafio 3 - Calculadora Simples

Desenvolver uma calculadora que permita realizar as operações:

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

O usuário informa dois números e escolhe a operação desejada.

**Conceitos praticados**

- Entrada de dados
- Operadores matemáticos
- Estruturas condicionais
- Lógica de programação
